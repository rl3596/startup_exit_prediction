# 教育与就业网络特征说明文档

**文件**: `edu_job_features.csv`
**行数**: 6,704 家公司（美国 AI 初创企业，融资总额 >= 100万美元）
**列数**: 15（1 个 ID + 14 个特征）

## 数据来源

这些特征基于两个二部图构建：

1. **人物-大学图**（52,554 个节点，68,773 条边）——来源于 `education` 表。每条边连接一个人与其就读的大学。
2. **人物-组织图**（135,965 个节点，190,461 条边）——来源于 `jobs` 表。每条边连接一个人与其工作过的组织。

大学和组织的重要性通过在这些二部图上计算的 **PageRank** 和 **度数** 来衡量。

## 标识符

| 列名 | 类型 | 说明 |
|------|------|------|
| `company_uuid` | ID | 公司唯一标识符。与 `feature_matrix.csv` 的关联键。 |

## 数据可用性标志

| 列名 | 类型 | 说明 |
|------|------|------|
| `edu_data_available` | 0/1 | 若至少一位创始人的教育数据已从 API 获取则为1。当值为0时，教育相关特征的可靠性较低（可能存在低估）。可作为模型中的控制变量使用。 |

## 教育网络特征

| 列名 | 类型 | 缺失率 | 说明 |
|------|------|-------:|------|
| `founder_top_univ_count` | int | 0% | 就读过**前20名大学**的创始人数量（排名依据为人物-大学图中的节点度数，即数据集中拥有最多创始人/相关人员的大学）。衡量创始团队的精英教育背景。 |
| `founder_univ_degree_avg` | float | 35.2% | 创始人就读的所有大学的平均度数。度数 = 数据集中就读该大学的人数。数值高意味着创始人就读的大学规模大、校友网络广泛，在创业生态系统中拥有众多校友。无创始人教育数据时为 Null。 |
| `founder_univ_pagerank_max` | float | 35.2% | 任一创始人就读大学中 PageRank 得分的最大值。PageRank 衡量递归重要性：当一所大学的校友本身与许多其他重要节点相连时，该大学得分更高。代表创始团队中（网络意义上）最"有声望"的学校。无创始人教育数据时为 Null。 |
| `co_alumni_investor_overlap` | int | 10.8% | 公司投资者中，其团队成员与至少一位创始人**就读过同一所大学**的投资者数量。衡量**通过教育关系形成的熟人连接**——创始人是否可能通过校友网络认识其投资者？每个投资者最多计算一次。公司无创始人记录时为 Null。 |
| `founder_alumni_network_size` | int | 35.2% | 数据集中与公司创始人就读过相同大学的其他人的总数（2跳可达：创始人 -> 大学 -> 其他校友）。衡量创始团队通过教育网络获得的**社会资本**。排除创始人自身。无创始人教育数据时为 Null。 |

## 就业网络特征

| 列名 | 类型 | 缺失率 | 说明 |
|------|------|-------:|------|
| `founder_ex_faang_count` | int | 0% | 曾在**大型科技公司**（Google、Meta、Amazon、Apple、Microsoft、Netflix、Nvidia、Tesla、OpenAI、Stripe、Uber、Airbnb 等约20余家）工作过的创始人数量。作为人才质量和知名企业经验的代理指标。 |
| `founder_ex_startup_count` | int | 0% | 曾在**数据集中其他公司**（即其他美国 AI 初创企业）工作过的创始人数量。衡量连续创业者或创业生态系统经验——创始人是否曾经历过完整的创业周期。 |
| `founder_prior_org_pagerank_max` | float | 38.3% | 创始人合并工作履历中任一组织的最大 PageRank 值（基于人物-组织二部图计算）。衡量创始人此前是否在就业网络中高度连接的重要组织工作过。无创始人工作数据时为 Null。 |
| `coworker_investor_overlap` | int | 10.8% | 公司投资者中，其团队成员曾与至少一位创始人**在同一组织工作过**的投资者数量。衡量**通过职业关系形成的熟人连接**——创始人是否可能通过前同事关系认识其投资者？每个投资者最多计算一次。公司无创始人记录时为 Null。 |
| `founder_coworker_network_size` | int | 38.3% | 数据集中曾与公司创始人在相同组织工作过的其他人的总数（2跳可达：创始人 -> 雇主 -> 其他员工）。衡量职业网络的广度。排除创始人自身。无创始人工作数据时为 Null。 |
| `founder_industry_diversity` | int | 0% | 所有创始人合并工作履历中不同组织的数量（排除当前公司）。衡量职业背景的多样性。数值越高表示创始人来自更多不同的公司，而非全部出自同一雇主。 |

## 跨图特征（教育 + 就业联合）

| 列名 | 类型 | 缺失率 | 说明 |
|------|------|-------:|------|
| `founder_investor_social_proximity` | int | 10.8% | 任一创始人与任一投资者团队成员之间**共享实体**（大学 + 雇主）的最大数量。例如，若一位创始人和一位投资者合伙人都曾就读斯坦福且都曾在 Google 工作，则共享数为2。衡量创始团队与其投资者之间最强社交连接的强度。0 = 未发现共同背景。公司无创始人记录时为 Null。 |
| `team_network_reach` | int | 23.6% | 通过校友网络和同事网络的**并集**可达的唯一人数（从创始人出发，经由大学和雇主的2跳可达）。衡量创始团队在所有关系类型中的综合社会资本。这不是校友网络规模 + 同事网络规模的简单相加，因为同一个人可能通过两条路径同时可达。无创始人教育和工作数据时为 Null。 |

## 缺失数据说明

缺失值出现在创始人教育或工作数据未能从 API 获取的情况下（约35-40%的人员数据缺失）。具体模式如下：

| 缺失模式 | 原因 | 受影响列 |
|---------|------|---------|
| 35.2% | 所有创始人均无教育记录 | `founder_univ_degree_avg`、`founder_univ_pagerank_max`、`founder_alumni_network_size` |
| 38.3% | 所有创始人均无工作记录 | `founder_prior_org_pagerank_max`、`founder_coworker_network_size` |
| 23.6% | 无教育数据且无工作数据 | `team_network_reach` |
| 10.8% | 公司完全没有创始人记录 | `co_alumni_investor_overlap`、`coworker_investor_overlap`、`founder_investor_social_proximity` |

对于树模型（XGBoost、LightGBM、HistGradientBoosting），NaN 值可原生处理，无需填充。对于其他模型，建议使用中位数填充并将 `edu_data_available` 标志作为控制变量。

## 模型效果

将这14个教育/就业特征加入基础网络 + 表格特征（31列）后：

| 模型 | 特征数 | CV ROC-AUC | 测试集 ROC-AUC |
|------|--------|------------|---------------|
| 仅网络 + 表格特征 | 31 | 0.7939 | 0.7854 |
| 网络 + 表格 + 教育/就业特征 | 45 | 0.7928 | 0.7999 |

ROC-AUC 提升 +0.015，幅度不大，可能与较高的数据缺失率有关。这些特征捕捉了投资图中不可见的社会资本和熟人关系连接。
